# 样片复核结论

上一轮粗拆解方向正确，但两个细节需要修正：

1. 书籍介绍段不是慢速横向轮播，而是书封快闪硬切。每张书封大约保留 4 到 8 帧，常见节奏约 0.13 秒一换。
2. 这些切点和音频瞬态有关。前 12 秒逐帧统计显示，书封段 14 个主要视觉切点里，多数和附近音频瞬态相差不超过 0.10 秒。

第一版采用的默认切点：

```text
134, 142, 146, 151, 155, 160, 164, 169, 173, 178, 182, 187, 191, 195
```

对应 30fps 下约：

```text
4.47s, 4.73s, 4.87s, 5.03s, 5.17s, 5.33s, 5.47s, 5.63s, 5.77s, 5.93s, 6.07s, 6.23s, 6.37s, 6.50s
```

实现取舍：

- 先做“卡点快闪书单 + 主书氛围页”。
- 音频先用示例节拍音频验证卡点。
- 封面先用参数化图形生成，后续可以替换成真实书封图片。
