# Claude Code 工作说明

## 项目目标

开发一个书籍类短视频模板生成器，先配置化，再产品化。

## 当前重点

当前处于阶段二：真实素材与配置化。

请优先保证：

- 配置文件能驱动书单、主书、字幕、开场视频。
- 封面查询脚本稳定可重复运行。
- Remotion 渲染不能因为缺封面或缺网络而失败。
- 第一阶段现有样片能力不能被破坏。

## 开发约束

- 使用 TypeScript。
- Remotion 动画用 `useCurrentFrame()`、`interpolate()`、显式帧号，不用 CSS animation。
- 大素材、缓存、输出视频不进仓库。
- 默认命令使用 Windows 兼容写法：`npm.cmd run ...`。
- 保持中文文档清晰，方便后续产品化。

## 建议开发顺序

1. 类型和配置读取。
2. 字幕轨道组件。
3. 封面查询与缓存脚本。
4. 开场视频替换组件。
5. 接入模板渲染。
6. 渲染验证。
