# v0.3 完成报告：素材库管理与客户端化体验

更新时间：2026-07-20

## 阶段目标

v0.3 的目标是把 `水星视频模板` 从“网页表单式编辑器”继续推进为“真正客户端式的视频模板软件”：

- 素材库可以承载多类素材，不再只做少量上传入口。
- 主制作界面从素材库中可视化选择素材，不再依赖手写路径或长下拉。
- 快闪书单的逐本封面素材可单独替换。
- 通用操作里的素材替换入口必须真实可用。
- 任务中心不再只是批量渲染队列，需集中展示最近生成的视频与批量任务。
- 导出配置和目录信息从主界面移到设置页。

## 本轮完成

### 素材库

- `/api/assets` 返回真实素材清单，并标记哪些素材可管理。
- 新增 `/api/assets/rename` 与 `/api/assets/delete`。
- 素材库页面支持分类筛选、搜索、预览、复制路径、改名、删除。
- 改名与删除不再依赖系统弹窗，改为页面内输入框与二次确认按钮，避免 Web / Electron 环境差异导致按钮无效。

### 主制作界面

- 背景音乐、快闪段背景图、主书封面、主书背景图、快闪书单逐本封面、开场视频、开场背景图均可从素材库选择。
- 素材选择改为可视化弹层，点击缩略图即可应用到当前素材位。
- 快闪书单仍保留文本批量编辑，同时每本书都有独立封面选择入口。

### 任务中心与设置页

- 任务中心展示最近生成的视频，并提供播放、下载、复制路径的 Web 降级操作。
- 批量任务记录仍保留在任务中心内，不再占用主制作界面。
- 设置页展示素材目录、输出目录和高级导出配置。
- Web 模式下不开放任意本地目录托管，避免扩大本地文件暴露面。

## 安全约束

- 删除与重命名只允许操作固定上传目录：
  - `public/covers/`
  - `public/backgrounds/`
  - `public/intro-videos/`
  - `public/audio/`
- 服务端拒绝 `..`、反斜杠、远程 URL、`data:`、`blob:`、跨分类路径。
- 重命名会清理文件名，并保留原扩展名，不允许把图片改成 `.html`、`.svg`、`.js` 等格式。
- 同名目标不会覆盖已有文件，会自动追加后缀。
- 已有加固继续保留：Host 校验、`Connection: close`、异步路由 `await`、上传白名单、批量 id 清理、`books` 非对象降级。
