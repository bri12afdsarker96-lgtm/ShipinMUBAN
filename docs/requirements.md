# 阶段需求

## 阶段一：卡点书封快闪 MVP

目标：验证样片拆解结论，并做出可运行的 Remotion 生成器。

已完成能力：

- 竖屏 `720x1280 / 30fps`。
- 前几秒情绪钩子场景。
- 约 `4.4s-6.5s` 的书封快闪段。
- 书封切点按 30fps 下约 4 帧一拍配置。
- 最终进入主书氛围页。
- 示例节拍音频用于验证画面卡点。

## 阶段二：配置化与真实素材

目标：把固定示例升级成可换素材、可换书单、可换字幕的模板生成器。

必须实现：

- 真实书封查询：优先 Open Library，补充 Google Books，网页查询只做兜底。
- 前几秒视频可替换：支持本地视频路径、裁剪起止、音量、静音、字幕覆盖。
- 字幕功能分离：用独立字幕轨道驱动开场、书封段和主书字幕。
- 中间跳转书籍可自定义：通过配置输入书名、作者、ISBN、本地封面覆盖。
- 最终选定书籍可自定义：主书、主封面、主背景、主字幕、进入时间点都可配置。
- 封面缓存：查询到的封面保存到本地，避免重复联网。

## 阶段三：产品化与批量生产

目标：让非开发用户也能通过软件批量生成书籍短视频。

规划能力：

- 本地可视化界面。
- CSV / Excel / JSON 批量导入。
- 自动卡点检测与手动微调。
- 模板库管理。
- 本地素材库和缓存。
- 渲染队列、失败重试、输出归档。
- 自动质检：缺封面、黑屏、字幕溢出、音频缺失、尺寸错误。
