# 三阶段产品化规划

## 目标

把配置化生成器升级成可给非开发用户使用的本地软件，支持批量生产书籍类短视频。

## 功能规划

- 可视化编辑界面：模板选择、书单编辑、封面预览、字幕编辑、开场视频选择。
- 批量导入：CSV / Excel / JSON，一行生成一条视频。
- 自动卡点：默认节奏、音乐瞬态检测、手动微调。
- 模板库：书籍情绪类、文学金句类、成长治愈类、名著推荐类、短剧感开场类。
- 素材库：封面、背景、开场视频、音乐、字幕样式。
- 渲染队列：排队、暂停、重试、失败原因、完成归档。
- 质检预览：缺封面、黑屏、字幕溢出、音频缺失、尺寸错误。

## 技术路线

- 渲染核心：Remotion。
- 本地服务：Node。
- 前端界面：React。
- 桌面封装：Electron 或 Tauri。
- 批量表格：CSV 优先，Excel 后续补充。
- 封面来源：Open Library / Google Books / 本地覆盖。

## 三阶段启动条件

- 二阶段配置结构稳定。
- 封面查询和缓存可用。
- 字幕轨道可独立配置。
- 开场视频可替换。
- 至少完成 3 条不同书籍主题的生成验证。
